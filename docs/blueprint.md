# **App Name**: W. Chloe Creative Hub

## Core Features:

- Talent Directory: Display a searchable directory of creative talent, with filters for category, location, and availability.
- Talent Profile Pages: Provide detailed profiles for each talent, showcasing their portfolio, bio, and contact information.
- Admin Dashboard: Enable administrators to manage talent profiles, blog content, and booking requests.
- Blog Manager: Allow admins to create, edit, and publish blog posts to highlight talent and agency news.
- Contact Form Processing: Process contact form submissions and store them for admin review.
- AI-Powered Talent Match Tool: Offer a tool to help brands discover talent. The AI tool takes search parameters such as desired aesthetic and budget, and then suggests several different types of creators or artist profiles.
- Secure Authentication: Implement secure authentication for both talent and administrators, with password reset functionality.

## Style Guidelines:

- Primary color: Muted teal (#73A580) evoking creativity and tranquility.
- Background color: Very light grayish-green (#F2F4F3), complementing the teal.
- Accent color: Soft lavender (#A6A1C3) adding sophistication and artistic flair.
- Headline font: 'Belleza', a humanist sans-serif to add a personal design-oriented feeling
- Body font: 'Alegreya', a humanist serif to add an elegant, intellectual, contemporary feeling
- Minimalist icons, with thin strokes, indicating categories, actions, and links.
- Clean and modern grid-based layout with ample whitespace.
